# ElizabethCSS
Microframework para el desarrollo web basando en TailwindCSS 

## Caracteristica

- Super ligera | ocupa menos de 32KB
- Modo oscuro permanente
- Una gran gama de colores
- Reset responsive
- Codigo reutilizable
